0. добавляем свою папку по фамилии
1. добавляем туда скрипт
2. добавляем программу с заданием. Так должно быть getuid(), getgid(), perror(), open()
3. первый коммит делаем так: git commit -am "commit message"
4. git push
5. при возникновении ошибки, то есть, если у вас не последняя копия, сначала надо будет сделать git pull, потом git push
